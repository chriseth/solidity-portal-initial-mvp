getting started

As a beginner, you find great tutorials, resources and tools that help you get started building with Solidity on the ethereum.org website.

Alternatively, learn the basics about blockchain, smart contracts and the EVM in the Solidity docs.

language design


We welcome Solidity power users, auditors, security experts and tooling developers to get involved and actively contribute the Solidity language design process.

  • Take part in the design discussions directly in the Github issues.

  • Join the solidity-users forum, where proposals for new language features and qualities of existing aspects of the language can be discussed.

  • Give input by completing (feature) feedback surveys which are regularly distributed via Twitter and the solidity-users forum.

  • Join dedicated language design discussion calls, in which one topic, issue or feature implementation is debated per call.
